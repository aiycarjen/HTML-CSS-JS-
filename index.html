HTML
บทที่ 1 — โครงสร้างพื้นฐาน HTML
HTML คือโครงสร้างของหน้าเว็บ เหมือนกระดูก ทุก element ถูกห่อด้วย tag เปิด-ปิด
<!DOCTYPE html> <html> <head> <title>หน้าเว็บแรกของฉัน</title> </head> <body> <h1>สวัสดีโลก!</h1> <p>นี่คือย่อหน้าแรก</p> </body> </html>
💡 จำง่ายๆ: tag ทุกตัวต้องมีปิด — เปิด <p> ต้องมี </p> เสมอ
HTML
บทที่ 2 — Tag ที่ใช้บ่อย
Tag สำคัญที่ต้องรู้จัก ได้แก่ heading, link, image, list
<!-- Heading h1-h6 --> <h1>หัวข้อใหญ่</h1> <h2>หัวข้อรอง</h2> <!-- Link --> <a href="https://google.com">ไป Google</a> <!-- รูปภาพ --> <img src="photo.jpg" alt="คำอธิบายรูป"> <!-- รายการ --> <ul> <li>รายการที่ 1</li> <li>รายการที่ 2</li> </ul>
CSS
บทที่ 3 — CSS คืออะไร?
CSS ทำให้หน้าเว็บสวยงาม — สี, ขนาด, ฟอนต์, ระยะห่าง ทุกอย่างคือ CSS
h1 { color: #6C63FF; font-size: 32px; text-align: center; } p { color: gray; line-height: 1.6; } .highlight { background: yellow; padding: 4px 8px; }
CSS
บทที่ 4 — CSS Box Model
ทุก element มี margin, border, padding และ content — นี่คือหัวใจของการจัดวาง
.box { width: 200px; height: 100px; padding: 16px; border: 2px solid #333; margin: 20px; background: #f0f0f0; border-radius: 8px; }
CSS
บทที่ 5 — CSS Flexbox
Flexbox ใช้จัดวาง element แนวนอน/แนวตั้ง ง่ายและยืดหยุ่นมาก
.container { display: flex; justify-content: center; /* แนวนอน */ align-items: center; /* แนวตั้ง */ gap: 16px; /* ระยะห่าง */ flex-wrap: wrap; /* ตัดบรรทัด */ } .item { flex: 1; /* ขยายเต็มที่ */ }
CSS
บทที่ 6 — CSS Grid
Grid ใช้จัดวางแบบตาราง เหมาะทำ layout หน้าเว็บทั้งหน้า
.grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 16px; } /* responsive */ @media (max-width: 600px) { .grid { grid-template-columns: 1fr; } }
JavaScript
บทที่ 7 — JavaScript เบื้องต้น
JS ทำให้เว็บ "ทำงานได้" — กดปุ่ม, แสดงข้อความ, คำนวณ ทุกอย่างเลย
let name = "สมชาย"; let age = 25; console.log("สวัสดี " + name); if (age >= 18) { alert("เป็นผู้ใหญ่แล้ว!"); } function greet(n) { return "สวัสดี " + n + "!"; }
JavaScript
บทที่ 8 — DOM: ควบคุมหน้าเว็บด้วย JS
DOM คือการเชื่อม JS กับ HTML ทำให้เปลี่ยนข้อความ, สี หรือซ่อน element ได้
let title = document.getElementById("myTitle"); title.textContent = "ข้อความใหม่!"; title.style.color = "red"; document.getElementById("myBtn") .addEventListener("click", function() { alert("กดปุ่มแล้ว!"); });
